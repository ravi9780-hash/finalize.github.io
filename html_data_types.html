HTML Data Types

In HTML, a data type is explained as the type of data that is used in the content of an element or in the value of an attribute.

In HTML Data Types Section, we will learn about the different data types classified under the following different specifications in separate chapters :

  • Basic HTML data types - HTML Basic Data Types
  • Data types defined by Request for Comments (RFC) and Internet Assigned Numbers Authority (IANA) documentation - RFC & IANA
  • Data types defined by World Wide Web Consortium (W3C) Specifications - W3C

Let's take a brief look at these.

HTML Basic Data Types

There are the following four basic data types in HTML :

  • character
  • text
  • name
  • number

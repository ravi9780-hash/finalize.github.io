HTML CHARACTER ENTITIES

Sometimes, we need to include some special character, such > and < in the content of an HTML element. These characters are called as entities. So, HTML enables us to include such symbols in the content of an element by using the various character entities. World Wide Web Consortium (W3C) calls the character entities as character encodings. There are some character, such as Greek and Chinese, which do not appear on the keyboard of the user. Therefore, you cannot write them directly from the keyboard. However, you can overcome form this problem by using the character entities in an HTML document. The character entity is composed of the following parts in the sequential manner :

  • Ampersand symbol (&)
  • Name of the entity
  • A terminating semicolon

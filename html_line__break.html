HTML LINE BREAK

Sometimes while writing the text, it may be required to have a line break at certain points. The BR element is used to define a line break in an HTML document.

HTML BR Element Attributes

The following table list the attributes of the BR element:

Attribute Purpose
class Defines the class name of element
id Defines the value of the id
style Defines an inline style of the element
title Provides an extra information about the element

HTML Line Break Example

Here is an example shows how to break a line using BR element in HTML.

<!DOCTYPE html>
      <html>
      <head>
         <title>HTML Line Break Example</title>
      </head>
      <body>
      
      <p>This is <br />
      HTML Line Break Example <br />
      shows how to break the line <br />
      using the BR element in HTML.</p>
      
      </body>
      </html>
